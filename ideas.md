## Selected Design Philosophy: **Dark Futuristic**

### Design Movement
**Neo-Brutalism meets Cyberpunk Minimalism** — Bold, confident, with technological edge. Inspired by cutting-edge AI platforms, developer tools, and forward-thinking tech companies.

### Core Principles
1. **Precision Over Decoration** — Every element serves a purpose; no unnecessary ornamentation
2. **Depth Through Contrast** — Dark backgrounds with vibrant accents create visual hierarchy and energy
3. **Technological Authenticity** — Design reflects the developer's expertise in cutting-edge tech
4. **Fluid Interactivity** — Smooth transitions and micro-interactions feel responsive and alive

### Color Philosophy
- **Primary Background**: Deep charcoal (`oklch(0.12 0.008 285)`) — professional, non-fatiguing
- **Accent Primary**: Vibrant cyan (`oklch(0.65 0.22 200)`) — represents innovation, AI, tech forward
- **Accent Secondary**: Electric purple (`oklch(0.58 0.25 290)`) — creativity, AI/ML sophistication
- **Accent Tertiary**: Neon lime (`oklch(0.72 0.20 130)`) — energy, success, highlights
- **Text Primary**: Bright white (`oklch(0.98 0.001 0)`) — clarity, contrast
- **Text Secondary**: Cool gray (`oklch(0.65 0.01 285)`) — hierarchy, subtlety
- **Card Background**: Slightly lighter charcoal (`oklch(0.18 0.01 285)`) — depth, separation

### Layout Paradigm
**Asymmetric Grid with Diagonal Flow** — Hero section uses full-width with diagonal accent stripe. Project cards are staggered in a masonry-like layout. Navigation is sticky, minimal, and always accessible. Sections flow with organic spacing, not rigid grids.

### Signature Elements
1. **Glowing Accent Lines** — Subtle cyan/purple glowing borders on key elements (cards, buttons, section dividers)
2. **Animated Code Snippets** — Small animated code blocks in hero and project sections to showcase technical expertise
3. **Gradient Overlays** — Subtle cyan-to-purple gradients on hover states and section backgrounds

### Interaction Philosophy
- **Hover States**: Cards glow with cyan border, slight lift (shadow increase), text accents brighten
- **Button Interactions**: Smooth scale and glow effect on hover; click feedback with brief pulse
- **Section Transitions**: Smooth fade-in and slide-up on scroll
- **Loading States**: Animated gradient pulses instead of spinners

### Animation Guidelines
- **Entrance Animations**: 400-500ms ease-out for cards and sections
- **Hover Effects**: 200ms smooth transitions for interactive elements
- **Micro-interactions**: 100-150ms for button clicks and state changes
- **Scroll Animations**: Staggered reveals (50-80ms between items)
- **Avoid**: Excessive motion; respect `prefers-reduced-motion`

### Typography System
- **Display Font**: `Space Grotesk` (bold, geometric, tech-forward) — headings, hero title
- **Body Font**: `Inter` (clean, readable, professional) — body text, descriptions
- **Monospace**: `JetBrains Mono` (code snippets, technical details)
- **Hierarchy**:
  - H1: 48px, Space Grotesk, bold, cyan accent
  - H2: 32px, Space Grotesk, semibold, white
  - H3: 24px, Space Grotesk, medium, cool gray
  - Body: 16px, Inter, regular, white
  - Small: 14px, Inter, regular, cool gray
